# Seven Diablos

什麼都靠打怪掉落，但怎麼打就是打不到。能不能只要努力農材料、打怪賺錢，就換獨特裝備穿？

本文設計了普通難度就能農到的材料，來合成各式裝備的赫拉迪克方塊公式，但又不至於三兩下就全身獨特裝備或成套裝備，享受有個目標可以追逐、稍加努力便能實現的樂趣～

為了農材料重複刷同樣關卡很乏味，覺得通關過程努力打怪賺錢就可買獨特裝備穿的話，也可向 NPC 購買材料來合成。

過去，你每次重玩《暗黑破壞神II》，大概只會練兩個最拿手的流派，就玩完收手了。現在，你會大呼過癮的把七種職業都練一次才收手，因為修改過後玩起來就是痛快！

## 下載與使用我的單機修改與赫拉迪克方塊公式

* 注意事項

  適用《暗黑破壞神 II：毀滅之王》1.13c 版和 1.14d 版，且創造新人物時必須勾選「資料片人物」，否則會當機。

  只要你有在玩 BATTLE.NET，就不要下載使用這個單機修改與新增公式，否則帳號會被封鎖！雖然玩 BATTLE.NET 時，不要下 -direct -txt 參數就沒事，但事實證明你總會忘記，不小心在 -direct -txt 參數模式下以修改過的設定玩 BATTLE.NET！

  所以本單機修改與合成公式，僅限只玩單機沒在玩 BATTLE.NET 的人使用！

* 內容

  data 資料夾裡面，存放各種單機修改，尤其是赫拉迪克方塊合成公式。

  _LOD_SharedStashSave.sss 是 PlugY 的倉庫檔，裡面放所有裝備，但這不是拿來穿的，而是當易牙居那樣的攻略網站查看裝備用的，有想要的裝備，再用合成公式做出來。

* 如何使用

  將 data 資料夾放在 Diablo II 資料夾，然後以 Diablo II.exe -direct -txt 啟動遊戲。不再使用這個單機修改與合成公式的話，將 data 資料夾刪除即可。

  將 _LOD_SharedStashSave.sss 檔案放到 [使用者資料夾]\Saved Games\Diablo II。

## 單機修改部分

年久失修，本節內容可能與實際修改有出入。

* 角色修改

  ```
  1. 跑步時不再消耗耐力。
  2. 一開始就有赫拉迪克方塊。覺得會影響劇情發展的話，可以丟到地上。
  ```

  不喜歡這些修改的話，可將 charstats.txt 刪除，不會影響其它單機修改部分。

* 商店修改

  ```
  ACT.1 阿卡拉賣赦免勳章，基德賣製作毒弓要用到的碎裂的綠寶石。
  ACT.2 卓格南賣綠寶石以外的碎裂的其它寶石，雷山德賣回復活力葯劑。
  ACT.3 奧瑪斯賣碎裂的骷髏，艾柯賣魔法詞綴寶石。
  ACT.4 賈梅拉賣符文石和全面回復活力葯劑，好好對決暗黑破壞神吧！
  ACT.5 安亞賣無瑕疵的寶石和骷髏。為什麼不賣完美的？因為合成符文石需要用到無瑕疵的，想要完美的，買三顆無瑕疵的合成就是了。
  ```

  除了赦免勳章，售價為遊戲預先的設定，並未更動。

  提升難度時，NPC 賣的最低階葯劑會升階，例如輕微治療葯劑變成輕型治療葯劑，導致最後 NPC 賣的整排葯劑其實都是同一種，所以改為 NPC 只賣一種治療葯劑和法力葯劑，越高章節賣的越高階。

  可以用合成公式轉換成套裝備和獨特裝備的符文石，是故意擺在 ACT.4 才賣。因為普通難度在對決暗黑破壞神之前，全身爛裝也能通關，只有這章沒準備好會卡關，所以這時擺上符文石方便合成獨特裝備。先好好享受打藍裝、轉黃裝穿的樂趣，等到正式對決暗黑破壞神時，更能感受改穿獨特裝備的驕傲。

* 怪物修改

  ```
  取消稀有 BOSS 的燃燒法力屬性。
  ```

  然而，隨劇情任務固定出現的 BOSS，屬性是寫死的，並不是隨機產生屬性的稀有 BOSS，所以依然有燃燒法力，例如 ACT 5 解救安亞時遇到的冰凍怪魔。

  不喜歡這項修改的話，可將 monumod.txt 刪除，不會影響其它單機修改部分。

* 物品修改

  ```
  1. 開啟天梯限定的符文組和獨特裝備。
  2. 同件獨特裝備可重複出現，而不是每場遊戲只出現一次。
  3. 獨特裝備、成套裝備、超強裝備，屬性效果都是最大值，不再浮動。
  4. 裝甲的防禦都是最大值，不再浮動。
  ```

  不喜歡超強裝備的修改，可將 qualityitems.txt 刪除，不會影響其它單機修改部分。

* 文字修改

  ```
  1. 修正簡體字造成的亂碼
  2. 以．符號取代•符號解決顯示為?的問題
  3. 符文石名稱後以圓弧顯示編號
  4. 寶石的文字顏色與符文石一樣為橘色
  5. 在中文版翻譯錯誤的「增加準確率」後面標上 (IAS) 以供辨識
  6. 攻擊時有 n% 機會施展等級 n 技能前面加上「被」
  7. 修正德魯依技能說明的錯別字
  ```

  不喜歡這些修改的話，可將 local 資料夾刪除，不會影響其它單機修改部分。

## 赫拉迪克方塊公式部分

  開啟天梯合成公式，並新增如下合成公式：

* 自訂公式

  ```
  辨視卷軸＝作弊大型護符
  物品＋辨視卷軸ｘ1~N＝作弊物品
  物品＋紅寶石＝普通變進階 進階變菁英
  物品＋藍寶石＝魔法物品
  物品＋黃寶石＝稀有物品
  物品＋綠寶石＝成套物品
  物品＋紫寶石＝獨特物品
  符石＋紫寶石＝約旦之石
  物品＋鑽石＝一般物品
  物品＋各種骷髏＝依骷髏等級打 2 到 6 凹槽（連獨特都能打到 6 凹槽）（附魔 2~3 凹槽，稀有、成套、獨特 1 凹槽，一般依物品等級 2~6 凹槽）
  物品＋生命葯劑＝修復
  鞋子＋體力葯劑＝跑速
  生命葯劑＋法力葯劑＝回復活力葯劑
  任意葯劑＝解毒葯劑
  ```

* 刪除與自訂公式重複的公式，保留之公式改為：

  ```
  數量ｘ3 改為數量ｘ2（無須耗材）
  相同寶石ｘ2＝高一級寶石
  相同符石ｘ2＝高一級符石
  回復活力葯劑ｘ2＝全面回復活力葯劑
  箭矢ｘ2＝十字弓彈
  十字弓彈ｘ2＝箭矢
  戒指ｘ2＝護身符
  護身符ｘ2＝戒指
  矛＋箭矢＝標槍
  斧頭＋匕首或短劍＝飛斧
  波形刀＋法杖＋腰帶＋法力葯劑＝長兵器
  劍＋法力葯劑＝吸血
  盾＋釘頭槌＋法力葯劑＝攻擊者受到傷害
  護身符＋法力葯劑＝所有抗性
  戒指＋投擲葯劑＝抗火
  戒指＋融解葯劑＝抗寒
  戒指＋回復活力＝抗電
  戒指＋解毒葯劑＝抗毒
  其它各樣耗材改為法力葯劑，例如手工藝品。耗材使用到葯劑的話，改為該葯劑，而不是法力葯劑。物品無須附魔。
  ```

## 推薦搭配 PlugY 進行遊戲

  PlugY 不只提供大倉庫，還提供許多遊戲設定，比如比照天梯模式，將 ACT.5 的凱恩移到傳送點附近，回城清理背包要找人鑑定時更順路，許多原本要在 data 資料夾檔案中修改的功能，它都能做到，建議使用：

  https://github.com/ChaosMarc/PlugY

  其實最好玩的，不是使用本 Seven Diablos 修改，而是單純用 PlugY 讓遊戲更像是天梯的遊戲方式和掉寶率！
